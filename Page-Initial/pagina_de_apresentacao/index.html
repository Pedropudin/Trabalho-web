Projeto Integrador: Este portal de e-commerce de dispositivos eletrônicos foi desenvolvido como trabalho final da disciplina Introdução ao Desenvolvimento Web, adotando uma arquitetura front-end modular.



Estrutura Semântica: Utilizamos HTML5 semântico para melhorar a acessibilidade (ARIA) e otimização de SEO, estruturando conteúdo em cabeçalhos (<h2>, <h3>), seções e listas.



Layout Responsivo: Aplicamos CSS3 com Flexbox e Grid para garantir fluidez em diversas resoluções (desktop, tablet e mobile), com media queries para ajustes finos de tipografia e espaçamento.



Interatividade: A lógica de front-end foi implementada em JavaScript puro, manipulando o DOM para validação de formulários, navegação suave (smooth scroll) e animações via @keyframes e transições CSS.



Boas Práticas: Adotamos Git/GitHub para controle de versão e revisão de código. Otimizamos performance com minificação de arquivos, carregamento assíncrono de scripts e lazy-loading de imagens.



Para dúvidas ou contribuições, abra uma issue ou pull request em nosso repositório no GitHub. Bom proveito e “eletrocurte-se”!
